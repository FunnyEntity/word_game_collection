# 数字游戏合集

一个包含数独、数织和数学挑战的网页游戏合集。

## 🎮 游戏介绍

### 数独游戏
- 经典的9x9数独游戏
- 自动生成可解的数独谜题
- 实时错误检查
- 支持键盘输入

### 数织游戏
- 根据数字提示填充格子
- 支持5x5、8x8、10x10三种尺寸
- 左键填充格子，右键标记格子
- 自动检查答案功能

### 数学挑战
- 随机生成加减乘除数学题目
- 计时和计分系统
- 难度逐渐增加
- 实时反馈

## 🚀 快速开始

### 本地运行
1. 克隆项目到本地
```bash
git clone https://funnyentity.github.io/word_game_collection/
```

2. 打开 `index.html` 文件
```bash
cd word-game-collection
open index.html
```

### GitHub Pages 部署
1. 将项目推送到GitHub仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择部署分支（通常是 main 或 gh-pages）
4. 访问 `https://funnyentity.github.io/word_game_collection/`
